Using the <line> Element

The <line> element is used to draw a line between two points. It has four attributes:

  • x1: The x-coordinate of the starting point.
  • y1: The y-coordinate of the starting point.
  • x2: The x-coordinate of the ending point.
  • y2: The y-coordinate of the ending point.

SVG path drawing, and the associated attributes

[02/03, 10:48 am] Meta AI: SVG path drawing is a powerful way to create complex shapes and designs. Here are the basic components of SVG path drawing and their associated attributes:

Path Commands

Path commands are used to define the shape of the path. The following are the basic path commands:

M (Move to):
Moves the starting point of the path to the specified coordinates.
L (Line to):
Draws a line from the current point to the specified coordinates.
H (Horizontal line to):
Draws a horizontal line from the current point to the specified x-coordinate.
V (Vertical line to):
Draws a vertical line from the current point to the specified y-coordinate.
C (Curve to):
Draws a cubic Bézier curve from the current point to the specified coordinates.
S (Smooth curve to):
Draws a smooth cubic Bézier curve from the current point to the specified coordinates.
Q (Quadratic curve to):
Draws a quadratic Bézier curve from the current point to the specified coordinates.
T (Smooth quadratic curve to):
Draws a smooth quadratic Bézier curve from the current point to the specified coordinates.
A (Arc to):
Draws an elliptical arc from the current point to the specified coordinates.
Z (Close path):
Closes the current path.

Path Attributes

Path attributes are used to customize the appearance of the path. The following are some common path attributes:

d (Path data):
Specifies the path commands and coordinates that define the shape of the path.
fill (Fill color):
Specifies the color used to fill the path.
stroke (Stroke color):
Specifies the color used to draw the path.
stroke-width (Stroke width):
Specifies the width of the path.
stroke-linecap (Stroke line cap):
Specifies the shape of the path ends.
stroke-linejoin (Stroke line join):
Specifies the shape of the path corners.
stroke-dasharray (Stroke dash array):
Specifies the pattern of dashes and gaps used to draw the path.
stroke-dashoffset (Stroke dash offset):
Specifies the offset of the dash pattern.
opacity (Opacity):
Specifies the transparency of the path.

Curve Attributes

Curve attributes are used to customize the shape of curves. The following are some common curve attributes:

rx (Radius x):
Specifies the x-radius of an elliptical arc.
ry (Radius y):
Specifies the y-radius of an elliptical arc.
x-axis-rotation (X-axis rotation):
Specifies the rotation of an elliptical arc around the x-axis.
